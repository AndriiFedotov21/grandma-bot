# Развертывание бота на Railway (за 2 минуты)

## Шаг 1: Подготовка

1. Создайте репозиторий на GitHub и загрузите туда код бота:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/ваш_username/бот_бабушка.git
git push -u origin main
```

2. Получите токен бота от @BotFather в Telegram (если еще не имеете)

## Шаг 2: Развертывание на Railway

1. Перейдите на https://railway.app
2. Нажмите "Login" и войдите через GitHub
3. Нажмите "New Project"
4. Выберите "Deploy from GitHub repo"
5. Выберите ваш репозиторий с ботом
6. Дождитесь завершения развертывания

## Шаг 3: Добавление переменных окружения

1. В Railway перейдите в Variables
2. Нажмите "Add Variable"
3. Добавьте:
   - Key: `TELEGRAM_BOT_TOKEN`
   - Value: ваш токен бота (из @BotFather)
4. Нажмите "Deploy"

Готово! Бот работает 24/7 и готов к использованию.

## Добавление бота в чат

1. Найдите вашего бота в Telegram по имени
2. Откройте его профиль
3. Нажмите "Добавить в чат"
4. Выберите групповой чат
5. Нажмите "Добавить"

Бабушка уже в деле!

## Если что-то не работает

Проверьте логи в Railway:
- Перейдите в Deployments
- Нажмите на последний деплой
- Откройте Logs

Если видите ошибку - сообщите мне!
