# Бабушка-бот для Telegram

Телеграм бот, который имитирует пошлую бабушку в групповом чате.

## Особенности

- Реагирует когда в чате упоминают бабушку
- Дает советы по-старушечьи когда звучат вопросы о политике
- Особое внимание Игорю (@Ihorushka) - много шуток про него
- Время от времени сама пишет пошлые шутки в чат
- Звучит как нормальный человек, без лишних знаков препинания

## Требования

- Node.js 14+
- Телеграм бот-токен (создается через @BotFather)

## Локальное запуск

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` с вашим токеном:
```
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

3. Запустите бота:
```bash
node index.js
```

## Развертывание на Railway

1. Перейдите на [railway.app](https://railway.app)
2. Нажмите "New Project" → "Deploy from GitHub"
3. Выберите ваш репозиторий
4. Откройте переменные окружения и добавьте:
   - `TELEGRAM_BOT_TOKEN` - ваш токен бота

Бот автоматически запустится и будет работать 24/7

## Развертывание на Render

1. Перейдите на [render.com](https://render.com)
2. Нажмите "New +" → "Web Service"
3. Подключите ваш GitHub репозиторий
4. Установите Runtime: `Node`
5. Установите Build Command: `npm install`
6. Установите Start Command: `node index.js`
7. Добавьте переменную окружения:
   - `TELEGRAM_BOT_TOKEN` - ваш токен бота

## Как получить токен бота

1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен

## Добавление бота в чат

После запуска бота найдите его в Telegram и добавьте в групповой чат.
